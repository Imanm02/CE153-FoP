# ورودی

ورودی شامل سه سطر است که در هر سطر سه عدد صحیح با فاصله از هم آمده است.
$$ a_{ij}\in \mathbb{Z}$$ $$-1000 \le  a_{ij} \le 1000  $$
 
# خروجی

خروجی برنامه شما باید دترمینان ماتریس ورودی باشد.



# مثال

## ورودی نمونه ۱
```
1 2 3
4 5 6
7 8 9
```


## خروجی نمونه ۱
```
0
```


## ورودی نمونه ۲
```
5 6 -3
2 1 7
-4 -2 6
```


## خروجی نمونه ۲
```
-140
```
